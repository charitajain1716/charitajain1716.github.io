IoT & Embedded Systems Internship

IIMSTC

Completed an internship in IoT and Embedded Systems where I worked on PCB design, embedded hardware development and component analysis. During the internship, I gained practical experience in schematic design, footprint assignment, PCB routing and Gerber file generation using KiCad. I also worked with Arduino and ESP32 based systems and learned how to analyze datasheets, select components and develop hardware solutions for embedded applications.

Projects Completed During Internship

RC Delayed LED Control Circuit

Designed and implemented an RC Delay LED Control Circuit to understand the charging and discharging characteristics of capacitors. The project demonstrates how a resistor-capacitor network can create a time delay before switching an LED ON or OFF. Different resistor and capacitor values were tested to observe their effect on delay timing and circuit performance. This project strengthened my understanding of electronic timing circuits, transient response analysis and practical circuit implementation.

Smart Blind Cane for Visually Impaired People

Developed a Smart Blind Cane to improve the safety and mobility of visually impaired individuals. The system integrates ultrasonic sensors for obstacle detection, IR sensors for path monitoring, water sensors for puddle detection and an accelerometer for fall detection. The device generates alerts through a buzzer and supports emergency notification features using IoT technology. This project combined sensor interfacing, embedded programming, real-time monitoring and hardware integration to create a practical assistive technology solution.
