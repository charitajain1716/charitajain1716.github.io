Gas Leakage Detection System

Project Description

Developed a Gas Leakage Detection System using Arduino Uno and an MQ gas sensor to detect harmful gas leakage in real-time. The system continuously monitors the concentration of gases and triggers a buzzer alert whenever the gas level exceeds a predefined threshold. This helps in preventing accidents and improving safety in homes, laboratories and industrial environments. Through this project, I gained practical experience in sensor interfacing, embedded programming, analog signal monitoring and real-time alert systems using Arduino.

Components Used

Arduino Uno
MQ Gas Sensor
Buzzer
LED
Breadboard
Jumper Wires
