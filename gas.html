Gas Leakage Detection System

Project Description

Developed a Gas Leakage Detection System using Arduino Uno and an MQ-series gas sensor to detect the presence of harmful gases in the environment. The system continuously monitors gas concentration levels and activates a buzzer whenever gas leakage exceeds a predefined threshold. This project enhances safety by providing early warning alerts and can be used in homes, laboratories and industrial environments. The project helped in understanding sensor interfacing, real-time monitoring and embedded system development.

Components Used

Arduino Uno
MQ Gas Sensor
Buzzer
LED
Breadboard
Jumper Wires
